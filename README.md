# DDos Go

A simple DDos CLI written with Go

# Installation

```console
$ go get github.com/iamstefin/ddosgo
```

# Usage

```
$ ddosgo --help
Usage of ddosgo:
  -n int
    	Maximum number of concurrent requests (default 200)
  -url string
    	The Target URL
```
